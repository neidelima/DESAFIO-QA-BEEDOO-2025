Contexto: Considerando que estamos analisando uma plataforma de cadastro de cursos, foi abordado em reunião junto ao time a implementação baseado nas dimensões funcionais e não funcionais utilizando a estrutura: "Como um , eu quero ,para que Gestor de Conteúdo (ou Administrador, Professor, dependendo de quem tem o privilégio do acesso) deve interagir com o sistema. Gestor o Conteúdo deve ter acesso a plataforma através de login e senha. Um novo curso deve ser cadastrado, para isso deve-se preencher um formulário de cadastro que fará o processamento/armazenamento dos dados. Input: O formulário de cadastro deve ser preenchido e validado nos 8 campos obrigatórios: Nome do Curso, Descrição do Curso, Instrutor, URL da Imagem de Capa, Data de Início, Data de Fim, Número de Alunos (Capacidade Máxima), Tipo de Curso (Ex: Presencial, Online). Output: O sistema deve registrar esses dados e exibir os cursos cadastrados em Listar Cursos. Validação dos Campos: 1. Nome do Curso, Descrição do Curso e Instrutor são obrigatórios. 2. Data de Início não pode ser anterior à data atual. 3. Data de Fim deve ser posterior à Data de Início. 4. URL da Imagem deve ter formato válido (Ex: .png, .jpg). 5. Número de Alunos deve ser um valor inteiro maior que 0. 6. Tipo de Curso são obrigatórios. 7. Na aba Listar Cursos deve haver as opções Excluir Curso/Editar Curso Layout do Formulário de Cadastro: O formulário de cadastro deve ser intuitivo, com rótulos claros e descrição completa para cada um dos 8 campos. Os campos obrigatórios deve conter asterisco (*). Deve haver um feedback visual claro (Ex: campo em vermelho) ao tentar submeter o formulário com dados inválidos. O botão "Cadastrar Curso" deve estar bem visível. Tempo de Resposta: O sistema deve cadastrar o curso e retornar a confirmação em menos de 3 segundos, garantindo uma boa experiência do Gestor de Conteúdo. Segurança: A rota/URL de cadastro de curso deve ser restrita apenas a usuários com o perfil de Gestor de Conteúdo. Disponibilidade: A funcionalidade de cadastro deve estar disponível 99.9% do tempo durante o horário comercial. 

Gestor de Conteúdo cadastrar um novo curso informando todos os dados necessários (Nome, Descrição, Instrutor, etc.) o curso esteja disponível para visualização e matrícula dos alunos na plataforma.
